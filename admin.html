📋 How to update the live website

1. Make your changes in any section using the sidebar.
2. Click 💾 Save Changes button.
3. Changes go live on https://baidstock-loans.vercel.app instantly — no redeploy needed!

✅ How it works

Changes are saved to Supabase database. The live website fetches content from the database automatically on every page load — so changes are instant.
